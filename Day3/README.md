\#Day 3 – Linux Basics for Cloud



Why Linux?



Most cloud servers run on Linux. You’ll need it for:



\- Managing servers (AWS EC2, GCP VM)

\- Using tools like Docker, Kubernetes

\- Automation with shell scripts
